# Tests

This directory contains all test and debug files for the Markdown Editor.

## Test Files

- `test_app.py` - Main application tests
- `test_image_paste.py` - Image paste functionality tests
- `test_incremental_preview.py` - Preview rendering tests
- `test_no_scroll_flash.py` - Scroll flash prevention tests
- `test_paste_functionality.py` - Paste functionality tests
- `test_scroll_preservation.py` - Scroll position preservation tests

## Debug Files

- `debug_image_paste.py` - Debug script for image paste issues
- `debug_paste_detailed.py` - Detailed paste debugging

## Running Tests

```bash
# Run from project root
python tests/test_app.py
```
